Gastty v0.7.1

🚀 First auto-update

This is the first release that installs itself — if you're already on v0.7.0, Sparkle handles the upgrade automatically (or click Gastty → Check for Updates… in the menu to grab it now).

🔧 Under the hood

  • The release-notes you're reading right now were generated by our new pipeline: .github/workflows/release-notes.yml renders the GitHub release body into a clean prefers-color-scheme HTML page hosted on GitHub Pages, which Sparkle's update dialog loads instead of the cramped GitHub release page.
  • The pipeline re-runs whenever a release body is edited — typo fixes, additions, screenshots all show up without re-tagging.
  • SPARKLE_PRIVATE_KEY is now strict-base64-normalised in CI so a stray paste artifact (like a zsh % end-of-line marker) can't break a release build.
  • Dependabot now watches the Sparkle Swift Package — within the 2.x line, Dependabot's PRs are safe one-click merges.
