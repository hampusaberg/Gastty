Gastty v0.7.2

What's new

  • Combined release notes when you've skipped versions. If you're a few versions behind, Sparkle's update dialog now shows notes for every release between you and the latest, not just the newest one. New page at /releases/index.html regenerates on every release publish/edit.
  • Stale-DMG nudge on first run. If someone passes around an older Gastty DMG (e.g. in Slack), the first launch now checks the appcast before onboarding starts and offers a one-click update to the current release. "Continue with this version" bypasses if you really want to.

Polish

  • Sidebar close animation no longer leaves "Connections" text floating. The sidebar's header label was anchored absolutely and kept drawing as the container shrank to 0; it now clips in step with the width animation, so ⌘S feels uniform.

Under the hood

  • Sparkle is gated off in Debug builds (Sparkle 2.x refuses to start without Developer ID signing — the resulting "updater failed to start" alert was pure noise during local development). Release builds are unchanged.
  • workflow_dispatch added to the release-notes workflow so the combined changelog page can be backfilled or recovered without re-editing a release.
