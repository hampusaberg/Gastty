Gastty v0.7.7

What's new

  • Saved credentials with macOS Keychain. Reusable SSH credentials (username + optional identity file + optional password) that you can attach to any saved connection by reference. Passwords live exclusively in the macOS Keychain (kSecClassGenericPassword), never in JSON. New Connections → Credentials… toolbar opens a table-style editor for managing them; the Connection editor gains a Credential picker row.
  • Right-click on connections in the sidebar to Edit or Remove. No more roundtrip through the Connections window for a quick rename or delete.
  • Automatic SSH password injection. When a credential has a stored Keychain password, sessions launch with SSH_ASKPASS_REQUIRE=force and a temporary mode-700 helper script — your password is fed to OpenSSH without an interactive prompt, even through a jumphost. The helper is deleted after 60 seconds.
