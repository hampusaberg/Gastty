Gastty v0.7.4

What's new

  • Settings → Keyboard. A new tab in Settings lets you rebind any of Gastty's menu shortcuts — New Window, New Tab, Close, Quick Connect…, Toggle Connections Sidebar, Connections…, Zoom In/Out, Actual Size. Click a shortcut to record a new combo; Esc cancels, Delete clears, the per-row undo arrow resets one binding, "Restore All Defaults" wipes everything.
  • Shortcut conflict diagnostics. If something on your system is claiming a keyboard shortcut Gastty wanted to use, the first window after launch now tells you what — and offers a one-click jump into System Settings to fix it. Two detection layers:
    • Exact: scans macOS App Shortcuts overrides and names the conflict precisely ("App Shortcuts remapped Quick Connect… to ⌘J").
    • Behavioural: notices when a Cmd-combo passes through but the matching menu action doesn't fire — surfaces "something else is intercepting this — possibly Karabiner-Elements, Raycast, BetterTouchTool, or Shortcat." (macOS has no API to name third-party event-tap apps, so this is the best we can do — but it's enough to point you in the right direction.)
