Gastty v0.7.10

What's new

  • Reset Terminal action under the View menu. When a TUI dies ungracefully (broken pipe, hard SSH drop) without sending the matching disable sequences for mouse reporting, alt-screen, or other modes it had enabled, the terminal stays stuck — most visible symptom is mouse moves leaking as ESC[<35;2;32M-style escape garbage typed into the local shell after the session dies. The new menu item injects reset into the focused tab to clear every mode flag back to defaults. No keyboard shortcut by default — assign one under Settings → Keyboard if you'd like.
