v0.7.6Thu, 28 May 2026

v0.7.6

Fixes

  • Keyboard shortcuts, cursor highlight, and Ctrl+W now survive an overnight sleep. v0.7.5 handled a quick pmset sleepnow, but a real deep sleep left three things broken until app restart: ⌘W / ⌘T would silently miss, the cursor stopped highlighting, and Ctrl+W typed gibberish (5u9;5u9;5u…) instead of deleting words. All three were symptoms of libghostty's focus state desyncing from AppKit's after deep sleep. Gastty now re-pushes surface focus to libghostty whenever a window becomes key, rebuilds the menu on both wake-from-sleep and app-becomes-active, and reinstalls its keyboard-event monitor after each wake to clear any stale CFRunLoop state.
v0.7.5Tue, 26 May 2026

v0.7.5

Fixes

  • Keyboard shortcuts stop working after the Mac wakes from sleep. ⌘W would close the whole app instead of the active tab/pane, ⌘T would pop a false "this shortcut is being intercepted" alert, and you'd have to quit and relaunch to recover. AppKit's menu dispatch can transiently fail after a long sleep — Gastty now subscribes to NSWorkspace.didWakeNotification, rebuilds the main menu so every menu item gets re-validated fresh, and pauses the conflict-detection logic for five seconds so the recovery transient doesn't trip a false alert.
v0.7.4Mon, 25 May 2026

v0.7.4

What's new

  • Settings → Keyboard. A new tab in Settings lets you rebind any of Gastty's menu shortcuts — New Window, New Tab, Close, Quick Connect…, Toggle Connections Sidebar, Connections…, Zoom In/Out, Actual Size. Click a shortcut to record a new combo; Esc cancels, Delete clears, the per-row undo arrow resets one binding, "Restore All Defaults" wipes everything.
  • Shortcut conflict diagnostics. If something on your system is claiming a keyboard shortcut Gastty wanted to use, the first window after launch now tells you what — and offers a one-click jump into System Settings to fix it. Two detection layers:
    • Exact: scans macOS App Shortcuts overrides and names the conflict precisely ("App Shortcuts remapped Quick Connect… to ⌘J").
    • Behavioural: notices when a Cmd-combo passes through but the matching menu action doesn't fire — surfaces "something else is intercepting this — possibly Karabiner-Elements, Raycast, BetterTouchTool, or Shortcat." (macOS has no API to name third-party event-tap apps, so this is the best we can do — but it's enough to point you in the right direction.)
v0.7.3Fri, 22 May 2026

v0.7.3

Fixes

  • Ctrl+C / Ctrl+D / Ctrl+W now work inside TUIs like Claude Code, lazygit, and helix. They already worked at the bash/zsh prompt; the bug only hit applications that negotiate the modern kitty keyboard protocol with the terminal. We were stripping the letter from the key event before libghostty could encode the proper CSI u sequence — now we follow upstream Ghostty's input convention exactly.

What's new

  • Confirm before closing a tab or pane with something running. ⌘W (or the tab's X button) now pops a "Close Tab?" sheet when the tab has an SSH session, vim, claude, or any other running process — so a stray keystroke doesn't kill your work. Idle shells still close instantly, so nothing changes for the common case. Splits get the same treatment with a "Close Pane?" sheet scoped to just the focused pane.

v0.6.1Thu, 21 May 2026

v0.6.1

🐛 Fix

  • Sharper text in split panes on Retina displays. The custom split view in v0.6.0 computed divider positions as raw floats — a 50/50 split on a 775pt bar landed the second pane at x=387.5, a sub-pixel origin that made the Metal-rendered text in that pane (and every nested pane after it) noticeably blurrier than the leftmost one. Rounding divider positions to whole points keeps every child origin on the pixel grid; every pane is now as sharp as the first.

v0.2.0

v0.1.0 Changelog

🚀 First release

The initial DMG. Gastty is a fast, GPU-accelerated macOS terminal built on libghostty, with a native Swift / AppKit shell.

What's in v0.1.0

  • GPU-accelerated terminal surface via libghostty (Metal)
  • Tabs with rename (double-click), drag-reorder, per-tab close, ⌘T / ⌘W / ⌘1–⌘9
  • Split panes — ⌘D (split right), ⌘⇧D (split down), ⌘[ / ⌘] to cycle, click-to-focus
  • Settings panel — font family, size, cursor style, theme, opacity, blur
  • Theme-aware chrome that tints with the active theme + opacity
  • 18 curated themes in the picker, 512 bundled
  • Find in scrollback (⌘F)
  • Session restore: tabs, splits, and working directories survive relaunch
  • Debounced OSC tab-title updates so zsh-theme bursts don't make tabs flicker

Download

Unsigned. Right-click → Open the first time to bypass Gatekeeper; subsequent launches go through normally.

  • SSH jumphost / ProxyJump support per connection (with the env-prefix workaround for libghostty's exec -l argv[0] handling that breaks -J)
  • Folders in the Connections settings page, with drag-reorder and per-folder connection assignment
