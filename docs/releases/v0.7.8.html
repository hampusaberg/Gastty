Gastty v0.7.8

Fixes

  • Paste no longer breaks on connections going through a ProxyJump host. Without explicit PTY allocation, certain SSH server configurations don't put the remote terminal into raw mode when the session is tunnelled via -J, which silently broke paste and a few other terminal modes. Gastty now passes -t alongside -J to force PTY allocation on the target.

Thanks @tomas-openinfra for the PR.
